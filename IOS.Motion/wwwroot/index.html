🔧 IOS.Motion 电机模块

✅ 服务状态

电机模块服务正在运行中...

📋 功能特性

电机控制 - 支持绝对位置和相对位置运动控制
状态查询 - 实时获取电机位置和运动状态
自动校准 - 支持电机原点校准功能
MQTT通信 - 通过MQTT协议接收和发布消息
配置验证 - 启动时自动验证配置参数有效性

🔗 MQTT 主题

订阅主题:

  • ios/v1/motion/control/move - 电机运动控制
  • ios/v1/motion/status/request - 状态查询请求
  • ios/v1/motion/calibration/start - 校准启动

发布主题:

  • ios/v1/motion/control/complete - 运动完成通知
  • ios/v1/motion/status/response - 状态查询响应
  • ios/v1/motion/calibration/complete - 校准完成通知
  • ios/v1/status - 服务状态通知

⚙️ 电机控制配置

以太网类型: CNet
最大阈值: 220,000
最小阈值: 0
脉冲比例: 1,000 pulses/unit

🏗️ 架构特性

  • 依赖注入: 使用 IOptions<T> 模式注入配置
  • 配置验证: 启动时自动验证配置有效性
  • 强类型配置: MotionControlOptions 提供类型安全
  • 模块化设计: 独立的配置扩展和验证器
  • 日志记录: 完整的 Serilog 日志支持
